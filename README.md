# ⚾ Baseball Game (Number Baseball) - Unreal Engine 5

언리얼 엔진 5를 활용하여 C++로 구현한 3자리 숫자 야구 게임 프로젝트입니다. 서버-클라이언트 구조에서의 판정 로직 구현과 데이터 동기화를 학습하기 위한 과제입니다.

---

## 🛠 프로젝트 개요

* **개발 환경:** Unreal Engine 5.5 / **JetBrains Rider** (Visual Studio에서 전환)
* **개발 언어:** C++
* **주요 기능:**
    * 서버 기반의 숫자 야구 판정 알고리즘 (Strike, Ball, Out)
    * **승리(3 Strike) 및 게임 리셋 시스템**
    * **PlayerState를 통한 플레이어별 시도 횟수 관리 및 동기화**
    * 클라이언트 입력값 검증 및 결과 멀티캐스트 알림 (채팅 및 UI)

---

## 🚀 주요 구현 내용

### 1. GameModeBase (게임 비즈니스 로직 및 흐름 제어)
* **판정 시스템 (`JudgeGame`) :** 클라이언트로부터 받은 숫자를 판정하고, 3 스트라이크 달성 시 승리 로직을 실행합니다.
* **게임 리셋 로직 (`ResetGame`) :**  `GenerateSecretNumber`를 통해 정답 숫자를 새롭게 생성합니다.
    * `AllPlayerControllers`를 순회하며 모든 플레이어의 `CurrentGuessCount`를 0으로 초기화합니다.
* **플레이어 관리 :** `OnPostLogin`을 통한 접속 관리 및 전송 위젯/공지사항 출력.

### 2. PlayerState & 데이터 동기화
* **시도 횟수 관리 :** 각 플레이어의 `ABGPlayerState`에서 `CurrentGuessCount`를 관리하여 개별 시도 횟수를 추적합니다.
* **프로퍼티 레플리케이션 :** 플레이어 번호 지정 및 주요 프로퍼티를 복제하여 멀티플레이어 환경에서 데이터 일관성을 유지합니다.

### 3. 통신 구조 (RPC)
* **Server_SubmitGuess :** 클라이언트의 입력값을 서버로 전달하고 유효성을 검사합니다.
* **Multicast_DisplayResult :** 서버의 판정 결과를 모든 클라이언트에 동기화하여 출력합니다.
* **채팅 시스템 :** RPC를 통한 메시지 입력 관리 및 출력 기능을 구현했습니다.

## 🔍 기술적 도전 및 해결 과정 (Troubleshooting)

### 1. UE 5.5 빌드 이슈 및 엔진 복구
* **문제 :** `PlayerState` 상속 클래스 추가 시 엔진 헤더에서 대량의 구문 오류 발생 및 빌드 도구(UBT) 실패.
* **해결 :** **에픽게임즈 런처의 '엔진 검증(Verify)' 기능**을 사용하여 손상되었거나 누락된 엔진 내부 파일을 복구함으로써 근본적인 빌드 이슈를 해결했습니다.
* **추가 조치 :** 이후 개발 환경을 Visual Studio에서 **JetBrains Rider**로 전환하여 프로젝트 안정성 및 코드 가독성을 높였습니다.

### 2. 승리/무승부 판정 및 데이터 초기화 로직
* **문제 :** 게임 종료 후 다음 라운드를 위한 데이터 초기화 및 정답 재생성 로직의 부재.
* **해결 :** `ResetGame` 함수를 구현하여 서버 사이드에서 정답 숫자를 갱신하고, 모든 플레이어의 상태 정보를 동시에 초기화하는 로직을 구축했습니다.

## 📂 클래스 구조 (Class Architecture)

| 폴더 | 클래스명 | 역할 및 주요 기능 |
| :--- | :--- | :--- |
| **Game** | `ABGGameModeBase` | 판정 알고리즘, **게임 리셋(`ResetGame`)**, 흐름 제어 |
| **Game** | `ABGGameStateBase` | 게임 전역 상태 및 공지 데이터 관리 |
| **Player** | `ABGPlayerController` | 사용자 입력 처리 및 서버-클라이언트 RPC 담당 |
| **Player** | `ABGPlayerState` | **개별 플레이어 데이터(시도 횟수 등) 관리** 및 동기화 |
| **UI** | `UNBWidget` | 숫자 입력, 판정 결과 및 공지사항 위젯 표시 |
